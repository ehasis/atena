Classe CAlien

Descrição:
 Classe derivada de CObjetoAvancado, serve para manipulação dos veículos aéreos inimigos. Tais objetos são dinamicos, é dizer se movem.

Propriedades:
SetarStatusSeta o status do alien.
ObterTipoRetorna o tipo do alien.
ObterStatusRetorna o status do objeto.
ObterEnergiaRetorna a energia.
ObterTirosRetorna um ponteiro do tipo CTiro para o objeto tiros do alien.
ObterMaisProximoEfetua o cálculo de distância entre dois pontos "d = ((x2 - x1)^2 + (y2 - y1)^2)^(1/2)" para localizar o alien da lista que está mais próximo do ponto informado.

Metodos:
CAlienSeta como NULL o ponteiro para o próximo nodo da lista.
CarregarArquivoDadosMétodo static para alocar o arquivo de dados que será usado por todos os objetos da classe.
DescarregarArquivoDadosMétodo desaloca a memória do arquivo de dados da classe.
ObterNumeroAliensRetorna o número de aliens adicionados.
IniciarSeta as variáveis conforme o tipo do alien.
DesenharDesenha o alien segundo o seu status e após chama o metódo para desenhar todos os tiros do alien corrente.
DesenharTodosChama o método Desenhar para todos os aliens que estão ativos.
AtualizarExecuta a movimentação e alteração do status do alien e após chama o método para atualizar todos os tiros do alien.
AtualizarTodosChama Atualizar para todos os aliens que colidirem com a área informada e que ainda estão ativos.
FinalizarLibera a memória de todos os aliens alocadas e tb chama o método de finalização dos tiros de cada alien.
AdicionarAloca memória para o novo alien e o insere no fim da lista.
ExcluirExclui todo o alien que corresponder as especificações informadas.
DecEnergiaDecrementa a energia pelo valor informado.
ChecarColisaoAliensRetorna verdadeiro se algum alien colidi com a área informada.
SonorizarSonoriza o alien com o som correspondente ao seu status e chama o método de sonorização dos seus tiros.
SonorizarTodosChama o método Sonorizar para todo o alien ativo.
SalvarAlienSalva todos os aliens da lista no final do arquivo informada, utilizando a estrutura TObjeto.
