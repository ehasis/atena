Classe CFilme

Descrição:
 Classe para manipulação de filmes.

Metodos:
ExecutarExecuta o script de filme passado como parametro.

Observações: O arquivo script passado como parâmetro possui modo de interpretação próprio, sendo definidos como:

Intruções do script
cls - limpa a tela
dat - carrega um arquivo .dat
mid - toca um midi
bmp - mostra um bitmap
txt - escreve um texto
seg - aguarta n segundos
fim - finaliza a execucao do script

Operadores
; - Comentarios
. - Cancela uma instrucao

Exemplo: 

------ abertura.txt --------
cls
dat filmes/abertura.dat
mid ABERTURA
bmp MULHER_01
txt Este texto sera exibido na tela
fim
----------------------------
