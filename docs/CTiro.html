Classe CTiro

Descrição:
 Classe derivada de CObjetoAvancado, serve para manipula todos os tiros do jogo

Propriedades:
SetarStatus - Seta o status do tiro.
ObterStatus - Retorna o status do objeto.

Metodos:
CTiroSeta como NULL o ponteiro para o próximo nodo da lista.
CarregarArquivoDadosMétodo static para alocar o arquivo de dados que será usado por todos os objetos da classe.
DescarregarArquivoDadosMétodo desaloca a memória do arquivo de dados da classe.
IniciarSeta as variáveis conforme o tipo do tiro.
AdicionarAloca memória para o novo tiro e a insere no fim da lista encadeada.
AtualizarExecuta a movimentação e alteração do status do tiro.
AtualizarTodosChama Atualizar para todos os tiros do objeto dono que colidirem com a área informada e que ainda estão ativas.
DesenharDesenha o tiro segundo o seu status.
DesenharTodosChama o método Desenhar para todos as tiros que estão ativos.
VerificarExisteTirosRetorna verdadeiro se existe tiros na lista encadeada.
ChecarColisaoTirosRetorna verdadeiro se algumo tiro está colidindo com a área informada.
SonorizarSonoriza o tiro com o som correspondente ao seu status.
SonorizarTodosChama o método Sonorizar para todo o tiro ativo.
FinalizarLibera a memória de todos as tiros alocados.
