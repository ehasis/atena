Classe GAScript

Descrição:
 Classe para interpretação de scripts.

Construtores:
GAScriptAlém do construtor padrão, há também mais 2 tipos, com ponteiros para para uma classe de eventos e lista de palavras reservadas.

Métodos:
FixarEventoDefine a classe da qual contém um método manipulador de eventos, sendo GAEventoScript::ExecutarInstrucao.
FixarPalavrasDefine a lista de de instruções a serem traduzidas pelo interpretador.
CarregarCarrega o arquivo de script.
ExecutarExecuta as instruções do script.

Observações:
Para utilizar o interpretador de script, a classe que o contém deverá ser derivada da classe virtual GAEventoScript, da qual contém o método ExecutarInstrucao.

O interpretador de script possui já duas palavras chaves reservadas, sendo elas iniciar (cod: 1000) e fim (cod: 1001). Das quais servem para devinir o ponto de interpretação e de finalização. Nota-se que a partir de 1000 os códigos relacionados as palavras chave estarão reservados para futuras implementações.

Exemplo: 

VETORPALAVRA lista;

lista.push_back(GAPalavra(1, "limpar"));
lista.push_back(GAPalavra(2, "escrever"));
lista.push_back(GAPalavra(3, "desenhar"));

GAScript script(this, lista);
script.Carregar("teste.gas");
script.Executar();
